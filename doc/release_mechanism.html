Releasing the tarball

  1. Update include/cdl_version.h
  2. Ensure your SVN tree is up-to-date: svn update .
  3. Ensure your SVN tree builds
  4. Ensure your changes are in: svn commit .
  5. Go to a new directory (in /tmp will do)
  6. Export the tree: svn export --quiet https://cyclicity-cdl.svn.sourceforge.net/svnroot/cyclicity-cdl/trunk cyclicity-
  7. Create a tar file: tar zcf cyclicity-.tgz cyclicity-
  8. Ensure the directory builds: Go to scripts, ln -s makefile.config. makefile.config, and go to bin and 'make'
  9. Upload the tarball: ftp upload.sourceforge.net, user anonymous, password empty; cd incoming; put cyclicity-.tgz
  10. Log in on SF as cdlsf
  11. Admin -> File Releases
  12. Add a file release of cyclicity-cdl; Release name is
  13. Select the file cyclicity-.tgz, and add file to the release
  14. Processor Any, File type source .gz, and update
Process complete - the file should be visible from Download->Browse All Files
